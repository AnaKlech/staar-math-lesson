# Підготовча подорож — STAAR Math G4

Interactive 5-day math practice lesson (Ukrainian), built from a student's past-test error
patterns mapped against an upcoming STAAR Math G4 test.

## Files

- `index.html` — page shell + styles
- `app.js` — app logic (progression, quiz state, review mode)
- `data.js` — lesson content (days, topics, questions)

All three files must be hosted together (e.g. GitHub Pages) — they reference each other.

## Hosting via GitHub Pages

1. Push this repo to GitHub.
2. Repo Settings → Pages → Source: deploy from `main` branch, root folder.
3. The lesson will be live at `https://<username>.github.io/<repo-name>/`.
